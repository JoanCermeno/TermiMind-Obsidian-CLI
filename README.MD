# 🧠 TermiMind

Un agente de IA local para documentar tus estudios y código directamente en Obsidian desde la terminal.

SOLO le dices a tu agente que te anote lo que quieras y te lo dejara con una marca de tiempo en obsidian.

### ✨ Características
- 🛡️ **100% Privado**: Funciona localmente con Ollama.
- ⚡ **Ultra Rápido**: Optimizado para modelos ligeros como Qwen2.5 1.7B.
- 📝 **Markdown Inteligente**: Crea tablas, emojis y callouts automáticamente.
- 🚀 **Instalación Global**: Un solo comando y listo.

### 🛠️ Instalación
```bash
git clone [https://github.com/tu-usuario/termimind.git](https://github.com/tu-usuario/termimind.git)
cd termimind
bash install.sh
